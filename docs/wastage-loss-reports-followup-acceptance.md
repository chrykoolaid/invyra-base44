# Wastage & Loss Reports Follow-up Acceptance Notes

Acceptance checks for the Reports follow-up:

- Reports can show all stock-out classes, not only wastage and store use.
- Loss type breakdown separates operational stock-outs from controlled loss lanes.
- Suspected loss, confirmed theft loss, and unknown shrinkage can be filtered by type.
- Pending/review values are separated from posted net values.
- Export includes review_required, review_decision, incident_reference, and evidence_reference.
- Existing Reports route remains stable through `ReportsTab.jsx`.

This is a reporting-only follow-up and should not change stock movement posting behavior.
